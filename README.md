# 🧮 Python Calculator

A simple, interactive command-line calculator built in Python that supports basic arithmetic operations.

---

## 📸 Preview

```
PS C:\Users\user1> 4+4
8
PS C:\Users\user1> 44+8
52
PS C:\Users\user1> 90*50
4500
PS C:\Users\user1> 600/50
12
```

---

## ✨ Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🔁 Continuous calculation loop
- 🚪 Easy exit option

---

## 🛠️ Requirements

- Python 3.x

No external libraries required — uses only Python's built-in modules.

---

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/python-calculator.git
cd python-calculator
```

### 2. Run the Calculator

```bash
python program1.py
```

---

## 📖 Usage

Once the program starts, you'll be prompted to select an operation:

```
Select operation:
1. Add
2. Subtract
3. Multiply
4. Divide
```

Enter two numbers when prompted, and the result will be displayed instantly.

> **Tip:** Enter `0` or the exit option to quit the program.

---

## 🐛 Known Issues / Fixed Bugs

- **ValueError on menu input**: Previously, entering expressions like `44+4` directly at the menu prompt caused a `ValueError: invalid literal for int() with base 10`. This was fixed by ensuring the menu only accepts valid integer choices.
